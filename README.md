## ğŸŒ Embed on Your Website

### Method 1: Direct Integration

```html
<!-- Add to your <head> -->
<link rel="stylesheet" href="path/to/style.css">

<!-- Add before closing </body> -->
<script src="path/to/script.js"></script>
```

### Method 2: iframe Embed

```html
<iframe 
  src="path/to/frontend/index.html" 
  style="position: fixed; bottom: 20px; right: 20px; width: 400px; height: 600px; border: none;">
</iframe>
```

### Method 3: Copy-Paste Widget

Just copy the chatbot HTML structure from `index.html` and include the CSS/JS files!
